# 🏦 Institutional Financial Risk Dashboard

A full-stack **financial risk analytics platform** built in Python using Streamlit.

This project models and visualizes key banking and investment risks including:
- Market Risk (VaR, Expected Shortfall)
- Credit Risk (PD/LGD simulation)
- Liquidity Risk (LCR)
- Capital Adequacy (Basel-style CET1)
- Quantitative Models (PCA, Correlation, Optimization, Machine Learning)

---

## 🚀 Features

### 📊 Core Risk Engine
- Monte Carlo stress testing
- Value at Risk (VaR 95%, 99%)
- Expected Shortfall (ES)
- Liquidity Coverage Ratio (LCR)
- Basel-style capital (CET1)

### ⚙️ Scenario Analysis
- Adjustable stress parameters:
  - Equity shock
  - Interest rate shock
  - Credit multiplier
  - Deposit run
  - Liquidity stress
- Preset scenarios:
  - 2008 Financial Crisis
  - COVID Shock

### 📁 Portfolio Input
- Upload custom portfolios via CSV
- Default sample portfolio included

### 📈 Quantitative Models
- Correlation matrix (diversification analysis)
- PCA (systemic risk drivers)
- Portfolio optimization (risk-adjusted allocation)
- Machine learning risk prediction

### 📉 Visualizations
- Loss distribution (VaR thresholds)
- Asset correlation heatmap
- PCA variance decomposition
- ML prediction scatter

### 📄 Export
- Generate downloadable PDF risk reports
